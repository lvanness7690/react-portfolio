Elevate Dashboard

Internal tools, technical operations, and AI-assisted development

Overview

Elevate Dashboard is an internal operations platform I built to centralize event planning, ticketing workflows, user administration, vendor and chef coordination, and production management in one shared system.

Problem

Operational information was spread across disconnected tools, manual processes, spreadsheets, and communication threads. The team needed a clearer source of truth, safer access controls, better exports, and repeatable deployment checks.

What I Built

  • React dashboard with serverless Node.js API routes on Vercel
  • Redis and Upstash-backed workflows for shared operational data
  • Google Sheets API reads and writes for ticket and operational workflows
  • User, invite, role, permission, event, ticket, and admin workflows
  • PDF and Word exports, production safety guards, and release checks

Technical Stack

React, JavaScript, Vercel, serverless Node.js API routes, Redis, Upstash, Google Sheets API, GitHub CI/CD, and Playwright smoke tests.

Impact

The dashboard turned fragmented workflows into a structured internal platform and showed the team how internal software and AI-assisted development could solve real operational problems quickly.
